# Project SPECTRE

This bot serves a file explorer and clearance system for Discord.

## Environment Variables

The application expects the following variables to be set in your environment or `.env` file:

- `DISCORD_TOKEN`: bot token for authenticating with Discord.
- `GUILD_ID`: ID of the Discord guild (server) where slash commands are registered.
- `MENU_CHANNEL_ID`: channel ID where the file explorer menu is posted.

Ensure these variables are available before running the bot.
